NRAO: National Radio Astronomy Observatory

  • Download Basics
    Overview:
  • This tool, called "DataVault Lite," provides direct download links for NRAO telescope data.
  • Provide a project ID to find a download cover page for the given project.
  • Development of this tool began in Jan 2009, and is on-going.
  • Proprietary data requires authentication, and in some cases, special request to the developers to provide access.
 
    Instructions:
  • See the homepage at /download.
  • Access projects directly by URL with your browser address bar.
  • See a full project with an exact project ID:
    e.g. /download/AGBT06B_023_07.html
  • See all projects with a partial match of the project ID:
    e.g. /download/AGBT06B_023.html, /download/AGBT06B.html
  • The format is: /download/project_name.html
 
    File types:
  • .tar.bz2 - bzip2 compressed tarball. On GNU/Linux, unpack with
    tar -xvjf filename.tar.bz2
  • .tar.gz - gzip compressed tarball. On GNU/Linux, unpack with
    tar -xvzf filename.tar.gz
  • .zip - ZIP file, recommended for Windows users.
  • .sdfits - GBT single-dish FITS file, sdfits
 
    Redirection:
  • In some cases, archived data cannot be provided directly through this tool, and you will be given a link to an external site which provides the requested data.
  • The goal of development is to provide a common look and feel for accessing all types of data. This is an on-going process.
